1.การเขียนภาษา C
    **ทุกคำสั่งจะจบด้วยเครื่องหมาย ;
    1.1)การประกาศตัวแปร
        -การตั้งชื่อตัวแรกต้องเป็นอักขระห้ามเป็นตัวเลข(ตัวพิมพ์ใหญ่พิมพ์เล็กถือว่าเป็นอักขระต่างกัน)
        -การใส่ประเภทข้อมูลหน้าตัวแปรเพื่อบ่งบอกว่าจะเก็บเป็นข้อมูลประเภทไหน
         EX. char ชื่อตัวแปร     (เก็บข้อมูลเป็นตัวอักษร)(1 byte)
             int ชื่อตัวแปร      (เก็บข้อมูลเป็นตัวเลข)(4 byte)  
             float ชื่อตัวแปร    (เก็บข้อมูลเป็นทศนิยม)(4 byte)
             double ชื่อตัวแปร   (เก็บข้อมูลเป็นทศนิยมหรือจำนวนเต็ม)(8 byte)
        -การใช้คีย์เวิร์ด long,short,unsigned
            long        (เพิ่มหน่วยความจำในการเก็บตัวแปรเป็น 2 เท่า)
            short       (ลดหน่วยความจำในการเก็บตัวแปรครึ่งหนึ่ง)
            unsigned    (เก็บตัวแปรเฉพาะค่าบวก)
    1.2)การใช้คำสั่ง printf()
        ตัวอย่าง code
            #include <stdio.h>

            int main()  
            {
                int x = 10;
                float y = 10.0;
                char z = "hello";

                //การแสดงค่าจำนวนเต็ม(%d)
                printf("%d",x);             >> 10

                //การแสดงค่าทศนิยม(%f)
                printf("%f",y);             >> 10.0

                //การแสดงค่าประเภทตัวอักษร(%s,%c)
                printf("%c",z);             >> hello
                printf("%s",z);             >> hello
            }

2.การคอมไพล์ภาษา C โดย MingGW
    2.1)การคอมไพล์
        ใช้คำสั่ง gcc ชื่อไฟล์.c
        EX. gcc hello.c
        ***โดย default จะได้ output เป็น a.exe

        ถ้าต้องการตั้งฃื่อไฟล์ output สามารถทำได้โดย
        EX. gcc hello.c -o hello
        จะได้ hello.exe
    2.2)การรันโปรแกม
        ใช้ ./ชื่อไฟล์
        EX. ./hello
